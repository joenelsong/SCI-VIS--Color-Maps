Image Differencer

This image differencing program can help you verify that your images are correct.

Directions:

  • Make a directory called "differencer"
  • Download the file differencer.cxx
  • The easiest way to do CMake is probably to adapt your own. I include some of my own here:
    • CMakeLists.txt for Mac
    • CMakeLists.txt for Linux
    • CMakeLists.txt for Windows
  • Run CMake in the differencer directory.
  • Modify differencer.cxx to point to the two images you want to compare.
  • Compile the program.
  • Run the program.
  • The program will (i) congratulate you, or (ii) tell you the images are not the same size, or (iii) output a difference map.
